Raised by Bill Murdock / Gage Krumbach: integrating off-the-shelf harnesses (OpenCode, Claude Code) into the agent catalog playground as a full chat interface requires session management, human-in-the-loop, and tool approval — "basically recreating Ambient." Not 3.5 scope (see [fact-agent-catalog-starter-kits.md](/features/agent-catalog/knowledge/fact-agent-catalog-starter-kits.md)), desired for 3.6.
