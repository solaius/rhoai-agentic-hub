The Jira home of the harness-compatibility and images work the catalog's
3.6 deploy story depends on: validating that off-the-shelf harnesses run
end-to-end against the platform's inference stack, and shipping slim,
license-clean container images for them. Relates to RHAIRFE-2443 (harness
images via AIPCC) and the
[AIPCC base-images fit question](/features/agent-catalog/knowledge/question-agent-catalog-aipcc-base-images-fit.md).
Naming note: predates the
[supported-not-validated ruling](/features/agent-catalog/knowledge/decision-supported-not-validated-images.md)
— its image tier should read "supported" downstream.
