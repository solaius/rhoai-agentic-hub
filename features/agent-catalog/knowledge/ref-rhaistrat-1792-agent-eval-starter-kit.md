The evaluation companion to the starter kits: reusable scorer templates
and a guided methodology so catalog agents can be assessed, not just
deployed. Grew out of the validated-agents/evals thread in the June UX
loop ([concept doc](/features/agent-catalog/knowledge/ref-validated-agents-evals-gdoc.md));
the optional "tested models" catalog field deliberately leaves room for
this to mature into a validation tier. Still in Refinement — not
release-committed.
