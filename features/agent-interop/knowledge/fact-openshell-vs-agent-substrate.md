Adel's framing (weekly report Jul 13-17, 2026):

- **OpenShell** solves **gate 1: security**. 88% of agent pilots fail
  at this gate. OpenShell provides secure-by-default sandboxing,
  identity, and policy enforcement.
- **Agent Substrate** solves **gate 3: scale/density**. Optimizes the
  infrastructure layer for running agents at scale.

These are **different layers** — OpenShell is the security gate that
agents must pass through before scale/density even matters.
