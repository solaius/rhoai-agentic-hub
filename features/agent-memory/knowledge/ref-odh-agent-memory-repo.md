`research/` and `product/` folders; starter issues for candidate
evaluations (Mem0 doc, OGX memory tool, file-vs-DB comparison);

product requirements and analysis land here, in the open — not in
private docs. Repo layout may be reshaped if MemoryHub is ported in with
commit history preserved.
