Status: Technology Preview in RHOAI 3.4. The "Kagenti" name is also used for the MCP Gateway Helm chart org (`ghcr.io/kagenti/charts/mcp-gateway`); the broader brand spans agent lifecycle management. See [ref-kagenti-agent-examples-repo.md](/features/agent-registry/knowledge/ref-kagenti-agent-examples-repo.md) for starter-kit examples, and [decision-agentops-control-plane-terminology.md](/features/platform/knowledge/decision-agentops-control-plane-terminology.md) for a related terminology clarification (OpenShift, not a separate "AgentOps control plane," is the control plane).

Superseded 2026-07-16: kagenti was removed from the Red Hat roadmap (2026-07-10). Upstream, the project remains active and is rebranding to Rosso (v0.6.1, June 2026) — the removal is a Red Hat decision, not an upstream wind-down. See [research/07-upstream](/features/agent-registry/research/07-upstream.md).
