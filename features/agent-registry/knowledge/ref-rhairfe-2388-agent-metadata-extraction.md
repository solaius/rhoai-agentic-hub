Automated, secure sync of agent capability metadata from running
workloads — the demand side of the card-verification story. Its strategy
clone (RHAISTRAT-1956, whose written approach was kagenti-implemented)
closed between 2026-07-11 and 2026-07-16 with no successor, leaving this
Approved RFE as the only live Jira expression of the requirement — and
the natural anchor for the post-kagenti re-file recommended in
[research/11-jira-gap](/features/agent-registry/research/11-jira-gap.md)
(see [fact-agent-registry-post-kagenti-discovery-gaps](/features/agent-registry/knowledge/fact-agent-registry-post-kagenti-discovery-gaps.md)).
