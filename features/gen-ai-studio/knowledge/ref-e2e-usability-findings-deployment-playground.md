Author: Yingzhao Zhou (UX Research). Critical findings: broken deployment-to-playground wayfinding, opaque deployment errors (forces OCP console), undiscoverable vLLM tool-calling config (5/5 users failed). High findings: confusing model-source identifiers, project filter hides content, undiscoverable playground features, opaque tool execution/creation. User needs: try-before-deploy, RAG beyond doc upload, MCP as a peer strategy to RAG, "validated" mental-model mismatch, HuggingFace as primary source. Fully incorporated into the old registry's `docs/knowledge-review/components/gen-ai-studio-architecture.md`.
