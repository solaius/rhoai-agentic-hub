MCP Catalog Knowledge Hub

The storefront of deployable MCP servers in Red Hat OpenShift AI (RHOAI)

The MCP Catalog is a curated catalog of MCP servers that you can discover, deploy, and manage directly on Red Hat OpenShift. Unlike discovery-only registries, the Catalog provides a governed path from discovery to consumption: discover in the catalog UI, deploy via the MCP Lifecycle Operator, connect through the MCP Gateway, and consume in Gen AI Studio.
