Jira Tracker (Strats Rollup)

Cross-component RHAISTRAT rollup for the MCP ecosystem. Every stored scope also matches RHOAIENG (and, for MCPLO, OCPSTRAT) delivery-level issues; only the RHAISTRAT-prefixed rows are tabled below, one section per component. Every Key, Type, Status, Fix Versions, and Updated value is sourced directly from that component's jira-snapshot.yaml. The Summary field is null for every row (the sweep's unauthenticated-probe rule), rendered as "(summary withheld)" below. The "Known As" column is a supplementary descriptor, not a Jira summary, populated only where an existing hub source names and describes the key; rows without a cited source show "-".

Gateway scope is provisional

The Gateway component's stored scope (project = RHAISTRAT AND summary ~ "\"MCP Gateway\"") is a first sweep with no explicit key list, unlike the other three components' scopes. It was validated against Jira (10 issues, all Feature type, no Outcome) but has not been reviewed by the feature owner for completeness - RHAISTRAT-1937 was the only key with an existing hub-cited description at sweep time. Treat the Gateway section below as a starting point pending owner refinement.
