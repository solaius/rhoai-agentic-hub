Provides OpenShift cluster management capabilities to AI agents/tools. Deployed via the MCP Lifecycle Operator (setup wizard or Day 2), not on by default; the standalone Helm chart path will be deprecated once the Lifecycle Operator reaches TP. No dependency on MCP Gateway (that comes later, for exposing/routing requests).

Entitlement and pricing-risk detail lives in the restricted counterpart of this entry.
