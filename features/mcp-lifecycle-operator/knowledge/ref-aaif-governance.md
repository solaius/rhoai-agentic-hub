The Agentic AI Foundation (AAIF), formed December 9, 2025 under the
Linux Foundation, governs MCP and related agentic AI projects.

**Founding projects:** MCP (Anthropic), goose (Block), AGENTS.md
(OpenAI). agentgateway (Solo.io) joined June 2026.

**Platinum members:** AWS, Anthropic, Block, Bloomberg, Cloudflare,
Google, Microsoft, OpenAI.

**Governance:** Three-tier -- Governing Board (strategic, chaired by
David Nalley/AWS), Technical Committee (one rep per Platinum member),
MCP Steering Group (merit-based Lead/Core/Maintainers). Seven working
groups including Identity and Trust, Accuracy and Reliability,
Transports, Agents, and Governance.

**Process:** Specification Enhancement Proposals (SEPs), formal SDK
tiers, published roadmap. The 2026-07-28 spec is the first major
release under AAIF governance.

**Events:** AGNTCon + MCPCon across NA, EU, Asia, India, Africa. First
MCP Dev Summit (Apr 2-3, NYC) drew 1,200 attendees.

Cross-cutting: MCP governance affects every MCP feature partition.
