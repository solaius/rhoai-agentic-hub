OCP team's feature for the MCP Lifecycle Operator General Availability.
Extends TP with production-grade capabilities: full CRUD lifecycle
management, automated rolling update strategies, clear status and
health reporting, automatic network policy setup, MCP Gateway
registration, Prometheus service monitor creation, and Agent Sandbox
(Code Mode) integration for network-isolated code execution.

Status: In Progress.
