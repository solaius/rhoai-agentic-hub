OCP team's feature for the MCP Lifecycle Operator Technology Preview.
Provides a standardized CRD-based API (MCPServer) for deploying and
managing MCP servers. Key goals: OpenShift RBAC integration, install
via RHOAI Limited, install-time server selection from catalog (with
per-tool granularity), Day 2 server additions, automated rollout
strategies for version upgrades.

Targeting mid-July 2026. Status: In Progress.
