RHOAI feature targeting General Availability for the MCP Lifecycle
Operator. Addresses the gap between TP-level support and production
readiness: dependency awareness (Gateway presence detection),
operational maturity (observability, error reporting), documentation
(deployment patterns, upgrade paths, runbooks), and full Red Hat
support commitment.

TP/DP target in RHOAI 3.5, GA target in RHOAI 3.6. Status: New.
