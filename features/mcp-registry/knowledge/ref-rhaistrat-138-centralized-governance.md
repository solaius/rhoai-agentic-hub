The original registry strategy feature: bring every MCP server through gated import, review and certification, RBAC-scoped visibility, and an auditable status lifecycle before anything reaches AI engineers. Closed — it seeded the governance core that shipped in the Dev Preview registry, and it anchors why registry = governance in this partition's strategy.
