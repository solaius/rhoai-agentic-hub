Unified UI in RHOAI structured as tabs — Catalog, Registry, Deployments — per asset type. AAA (AI Available Assets) is the RBAC-scoped consumption surface for AI Engineers. All RHOAI UI and prototypes use PatternFly (Red Hat's open source design system) for components, design tokens, and layout patterns — see [ref-patternfly-design-system.md](/features/platform/knowledge/ref-patternfly-design-system.md).
