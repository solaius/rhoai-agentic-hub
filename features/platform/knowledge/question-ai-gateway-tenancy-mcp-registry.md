The AI Gateway F2F decided the existing MaaS group-based tenancy model generalizes to MCP tool catalogs (decision 5 in [decision-ai-gateway-f2f-architecture.md](/restricted/features/platform/knowledge/decision-ai-gateway-f2f-architecture.md)), but the implementation details of how that tenancy boundary composes with the MCP Registry's own governance model (see [fact-mcp-registry-data-model-proposal.md](/features/mcp-registry/knowledge/fact-mcp-registry-data-model-proposal.md)) are still undefined.
