The first systematic benchmark for evaluating agent skills quality.
Scored 47,150 public skills on a 12-point scale. Key findings: average
quality 6.2/12, 73% carry elevated safety risk, curated skills raise
pass rates by +16.2pp, self-generated provide negligible benefit
(-1.3pp), focused 2-3 module skills outperform larger bundles (+18.6pp
vs +5.9pp). Provides evidence that quality and focus beat catalog size.

Paper: arXiv 2602.12670. Website: skillsbench.ai
