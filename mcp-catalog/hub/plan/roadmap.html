Roadmap

The MCP Catalog's own release plan across RHOAI versions: Dev Preview shipped in 3.4, Dev Preview continues through 3.5, Tech Preview targeted for RHOAI 3.6 EA1, and GA targeted at RHOAI 3.6 stable.

Version Timeline

Version Release Key Capabilities
3.4 DP Shipped Apr 2026 (code freeze Apr 10, 2026) 5 partners + 2 community MCPs, deploy wizard, manual prerequisites (10 servers shipped total; see What Is the MCP Catalog?)
3.5 (DP continues) DP continues Aug 2026 Enhanced metadata; the Catalog itself stays in Dev Preview stage this release. Registry integration work is re-targeted to RHOAI 3.6 EA1 (see source note below), so it does not land in 3.5.
3.6 EA1 TP TP 2026 (date TBD) Catalog reaches Tech Preview alongside MCP Registry TP. Prerequisite automation ships here (RHAIRFE-2434, Approved) as part of the Catalog-Registry integration, delivery Feature RHAISTRAT-2027 (In Progress, fix version 3.6 EA1 RHOAI RELEASE), which supersedes the obsoleted RHAISTRAT-1938.
GA GA RHOAI 3.6 (code freeze Oct 23, 2026) Multi-registry federation, partner pipeline scaling, tier labeling (RHAISTRAT-1994, status New per this feature's Jira sweep).

Owner re-plan (2026-07-10/11): MCP Registry Dev Preview will not land in RHOAI 3.5 stable. The current push is MCP Catalog TP and MCP Registry TP together in RHOAI 3.6 EA1, alongside the Catalog-Registry integration row above. This re-plan is why prerequisite automation (RHAIRFE-2434) and the Catalog-Registry integration both moved from an earlier RHOAI 3.5 target to RHOAI 3.6 EA1.

RHAIRFE-2434 dating: the Jira-sourced management hub plan/jira-tracker.html and the Open Questions section of plan/gaps.html (Q7) tie RHAIRFE-2434 to RHOAI 3.6 EA1 via delivery Feature RHAISTRAT-2027, confirmed In Progress with fix version 3.6 EA1 RHOAI RELEASE in this feature's own jira-snapshot.yaml (swept 2026-07-11). This page follows the Jira-verified 3.6 EA1 dating, consistent with Enablement & Troubleshooting.

Catalog-Registry integration re-scoped (2026-07-10): RHAISTRAT-1938 ("[TP] MCP Catalog and Registry Integration - Unified Discovery and Governance for Tech Preview") is now Closed/Obsolete (confirmed in the snapshot), superseded by RHAISTRAT-2027 ("MCP Catalog and Registry Integration - Unified Discovery-to-Governance Workflow for Tech Preview," In Progress), which carries fix version 3.6 EA1 rather than 3.5.

Dependencies

Catalog delivery hangs off the same umbrella outcome as the rest of the MCP ecosystem, RHAISTRAT-1339 (AI Hub AI Asset Delivery for Agentic Solutions); see Jira Tracker for the Catalog's own tracked slice of that outcome. The Catalog depends on the MCP Lifecycle Operator for its deployment primitive and on the MCP Registry for the governance and federation work targeted at 3.6 EA1 and GA; neither dependency is Catalog-owned, so their own schedules can move the Catalog's TP and GA dates.
