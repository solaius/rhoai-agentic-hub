From Adel Zaalouk's blog ("Why agentic AI needs an open inference stack"):

**The paradox:** Gartner projects a 90% reduction in per-token costs by
2030. Goldman Sachs forecasts a 24x increase in token consumption by 2030.
Net result: aggregate enterprise spend rises despite cheaper units. This
is the Jevons Paradox applied to AI inference — efficiency gains unlock
new agentic use cases that consume far more than the savings.

**Supporting data points:**
- One OpenClaw contributor consumed $1.3M in OpenAI API tokens in one
  month — 603B tokens across 7.6M requests from ~100 Codex instances.
  Single-day peak: $19,985.84.
- Jensen Huang: agent token consumption ~1,000x vs traditional usage.
- Marc Andreessen: $1,000/day on Claude tokens running agents; latent
  demand $5,000-$10,000/day per fully deployed personal agent.
- AI companies currently spend ~195% of revenue (isaiprofitable.com).
- GPU supply locked 3-4 years ahead.

**Implication for the narrative:** self-hosted inference isn't a cost
optimization — it's a structural necessity as agents scale. This is the
core economic argument connecting the Inference pillar to the Agents
pillar.
