**When** I'm exploring agentic AI for my organization,
**I want to** build my first agent with Red Hat AI,
**so I can** validate the platform fits my use case before committing.

**How RHOAI addresses this:**
- GenAI Studio playground for experimenting with models, prompts, and tools
- Agent templates (first form factor — configurable code agents from any
  framework, surfaced in AI Hub)
- Starter kits for common patterns (RAG agent, tool-calling agent, etc.)
- AI Hub as the discovery plane for all AI assets

**Pillar:** Agents (BYOA theme + AI Hub theme)
